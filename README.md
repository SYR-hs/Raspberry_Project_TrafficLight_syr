# 예외처리

프로그램을 실행하다 보면 입력 오류, 파일 없음, 형변환 실패 등과 같은 상황이 자주 발생한다.

이때 프로그램이 멈추지 않도록 오류를 처리하고 정상 흐름으로 이어가게 하는 것을 '예외처리'라고 한다.

여기서 예외는 프로그램 실행 중 발생하는 오류 상황을 말한다.

파이썬에서는 try, except 구문을 사용한다.

+ 예외 처리의 목적은 크게 두 가지로 나뉜다.

-> 프로그램이 갑자기 종료되지 않게 하기

-> 오류 상황에서 사용자에게 안내하고 대체 동작을 수행하기

# try / except 기본 구조

+ 예외가 발생할 수 있는 코드를 try 에 작성, 오류가 났을 때 실행할 코드를 except 에 작성.

try:
    # 오류가 발생할 수 있는 코드
    x = int(input("숫자를 입력하세요: "))
    print("입력한 값:", x)
except:
    # 오류가 발생했을 때 실행할 코드
    print("숫자가 아닙니다.")

try 안에서 오류가 나면, 아래로 더 실행하지 않고 except 로 이동한다.

오류가 없다면 except는 실행되지 않는다.

+ finally 는 오류가 나든 안 나든 항상 실행된다.

파일 정리, 네트워크 종료, 마무리 출력 같은 '정리 작업'에 자주 사용한다.

+ else 는 예외가 없을 때 즉, try 에서 오류가 발생하지 않은 경우 실행된다.
